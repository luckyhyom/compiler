# compiler
node.js는 런타임이다.
런타임: 실행환경

node.js를 이용하면 터미널에서 js파일을 실행시킬 수 있다.
이는 터미널에서 java파일 등을 실행시킬 수 있는 것과 같다.

결국 js파일이나 뭐나 다 메모장에 적어놓고 실행시키는거랑 같은 거였다.

결론: 본래 웹브라우저에 연결하여 (html파일?) 실행되던 js를 터미널환경에서도 실행할 수 있게 됐다. ⇒ 다른 환경에서도 실행 가능
